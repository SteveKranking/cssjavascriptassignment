Javascript Basics

Javascript is a cross-platform, object oriented language. JAvascript is extremely popular for a variety of reasons. It is small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. Javascrit lives inside a host environment (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.
